# Experimental Assistant v0.9.9 Release Notes

This is the version-specific release record for v0.9.9. The maintained full notes are in [`V0_9_RELEASE_NOTES.md`](V0_9_RELEASE_NOTES.md) and are included in the release manifest.

## Version

- Package version: `0.9.9`
- Release label: `v0.9.9`
- Distribution: `experimental-assistant`
- CLI: `ea`
- Public Codex skill: `$ea` only

## Release Scope

- Universal schema-driven literature data collection with the six former electrical categories retained as templates.
- Keyed Chinese/English dynamic report localization across eight methods.
- Figure-local source data, opt-in audit appendix, and readable report-bound image identifiers.
- A resumable read-only first-project journey.
- Retired Compatibility skill absent while historical readers remain supported.
- Candidate-bound automated, public-benchmark, Mock, simulated-agent, and manual-artifact evidence.

## Relationship To v1.0

v0.9.9 is the final planned feature update before v1.0. Real-user trials, external expert sign-off, and live-account Zotero testing are optional post-v1 improvement activities, not promotion gates. The release dossier must state evidence types and limits without implying real-user or independent-expert validation.
