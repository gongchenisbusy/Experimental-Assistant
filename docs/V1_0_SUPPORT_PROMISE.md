# Experimental Assistant v1.0 Support Promise

The v1.0 public surface is the `experimental-assistant` distribution, `ea` CLI, and single `$ea` Codex skill. It supports Python 3.11–3.13 on Windows, Ubuntu, and macOS, subject to the public CI and release records.

EA promises protected imports, explicit review gates, provenance and reference records, recoverable project lifecycle operations, guided first-project routing, concrete errors/next actions, and checksum-verifiable exports. It preserves v0.9.7/v0.9.8 project and protocol readers where documented.

EA does not promise autonomous scientific proof, exhaustive literature coverage, universal PDF/OCR/table extraction, automatic cross-paper comparability, access-control bypass, credential management, or stable live-account behavior for optional browser/Zotero/institution integrations. Evidence types and limits remain public.

Real-user trials and external expert reviews may inform later releases, but they are not hidden prerequisites for the v1.0 support contract.
