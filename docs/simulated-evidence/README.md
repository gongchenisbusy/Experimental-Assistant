# v0.9.9 Simulated Release Evidence

This directory contains release-candidate evidence that can be produced without real users, external experts, private data, live Zotero accounts, or institution credentials.

Allowed `evidence_type` values are `automated_test`, `simulated_agent`, `mock_fixture`, `public_benchmark`, and `manual_artifact_review`. Every record states its candidate commit or tree, inputs, result, findings, limits, and retest status. A simulated-agent or Mock record must never be described as real-user testing or independent-expert validation.

The final release dossier is `docs/V1_0_READINESS_DOSSIER.yml`.
